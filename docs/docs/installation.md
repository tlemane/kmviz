# Installation

## Using pip

```bash
pip install kmviz
```

## From sources

### Clone the repository
```bash
git clone https://github.com/tlemane/kmviz.git
```

### Install

Requires `python` and `poetry`.

```bash
cd kmviz
poetry shell
poetry install
```

### For nix users

```bash
cd kmviz
nix develop
poetry shell
poetry install
```
